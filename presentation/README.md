# GIX — Presentation

A self-contained slide deck and per-slide speaker scripts for presenting the GPU Inference Exchange.

## Contents
- **[`presentation.html`](presentation.html)** — the deck. Open it in any browser (or host via GitHub Pages). 10 slides, light "Palantir" theme, keyboard-navigable.
- **[`scripts/`](scripts/)** — one Markdown speaker script per slide (full narration + on-screen cue + duration + transition line).

## Presenting
- **Navigate:** `←` / `→` (or click the arrows / dots), `Home` / `End` to jump.
- **Speaker notes:** press **`N`** (or the *Notes* button) for a condensed cue per slide; the **full** script for each slide is in `scripts/NN-*.md`.
- **Fullscreen:** press **`F`**.
- Total runtime is ~12 minutes at a steady pace.
